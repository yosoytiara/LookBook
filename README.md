## Technologies

- Frontend: React, TypeScript, CSS
- Backend: Express
- Database: MongoDB
- Tools: Webpack

## Installation

1. Fork/Clone the repository:

   ```bash
   git clone
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the backend server:

   ```bash
   cd backend
   node server.js
   ```

4. Start the frontend server:
   new terminal
   ```bash
   cd ..
   npm start
   ```
